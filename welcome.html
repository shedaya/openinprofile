Setup Steps
1

Install the extension ✓ Done

You've already completed this step — the extension is loaded and running.

2

Download & run the companion app

Because Chrome can't open a different profile on its own, a tiny helper app does that work. It runs silently in the background only when you use the extension.

What to do:

1. Download the companion app from GitHub — grab open-in-profile.zip from the latest release and unzip it.

2. Unzip it. Put the native-host folder somewhere permanent — like C:\OpenInProfile\. Don't move it after this.

3. Double-click install.bat inside that folder. It will ask for your Extension ID (shown below) and handle everything else automatically.

3

Set up your profiles

Tell the extension which Chrome profiles you have. You only do this once — the settings apply to every profile automatically.

Click Auto-detect my profiles in the settings page to find your profiles automatically — no need to look anything up manually.

How to install

1. Download open-in-profile.zip from the latest release and unzip it.

2. In Chrome, go to chrome://extensions → enable Developer mode → Load unpacked → select the extension folder.

3. Move the native-host folder somewhere permanent (e.g. C:\OpenInProfile\) and double-click install.bat. Paste your Extension ID when asked.

4. Click the ⇄ toolbar icon → Manage profiles → Auto-detect my profiles → Save.

Privacy Policy

Open in Profile — Privacy Policy

Open in Profile does not collect, transmit, or store any personal data of any kind.

The extension communicates exclusively with a locally installed companion application on your own computer. No data is sent to any external server, third party, or remote service.

Your profile names and preferences are stored in a single local file at %APPDATA%\OpenInProfile\profiles.json on your PC. This file never leaves your computer.

The extension requests the following Chrome permissions:

contextMenus — to add the right-click menu items.
tabs — to read the current tab's URL when you click the toolbar button.
nativeMessaging — to communicate with the local companion app.
scripting — to show error messages if the companion app isn't installed.

No analytics. No tracking. No ads. No accounts. No data of any kind leaves your device.
